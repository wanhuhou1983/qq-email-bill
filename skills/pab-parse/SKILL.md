# 平安银行账单解析器

## 用法
```bash
python scripts/pab_parse.py <文件或目录> [output.xlsx]
```

## 结构
- **输入**: HTML 文件（fetched3 目录）
- **公式**: 本期应还 = 上期账单 - 上期还款 + 本期账单 - 本期调整(含退款) + 循环利息
- **方向**: 正数=支出，负数=收入/退款
- **特点**:
  - 按卡分组：每张卡有独立名称和"合计"小计
- 双重校验：公式 + 每卡合计 vs 交易净额
- 卡号提取：平安XX卡（NNNN）格式

## 数据源
- 文件目录见脚本内默认路径
