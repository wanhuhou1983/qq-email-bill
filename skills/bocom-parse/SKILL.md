# 交通银行账单解析器

## 用法
```bash
python scripts/bocom_parse.py <文件或目录> [output.xlsx]
```

## 结构
- **输入**: HTML 文件（fetched3 目录）
- **公式**: 本期应还款 = 上期应还款 - 还款/退货/费用返还 + 消费/取现 + 费用 + 利息
- **方向**: 正数=支出，负数=还款/退货，金额带 CNY 前缀
- **特点**:
  - 收入表（还款、退货、费用返还明细）和支出表（消费、取现）分两个 <table>
- 三重校验：公式 + 收入合计 vs 汇总 + 支出合计 vs 汇总
- 支持 RMB/USD 双币

## 数据源
- 文件目录见脚本内默认路径
