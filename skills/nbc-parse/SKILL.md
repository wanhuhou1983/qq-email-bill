# 宁波银行账单解析器

## 用法
```bash
python scripts/nbc_parse.py <文件或目录> [output.xlsx]
```

## 结构
- **输入**: PDF 文件（宁波银行信用卡账单YYYYMM.pdf）
- **公式**: 上期账单金额 - 本期还款金额 + 本期新增金额 + 本期调整金额 + 本期循环利息 = 本期账单金额
- **方向**: 正数=消费，负数=还款/入账
- **特点**:
  - PDF 解析（pdfplumber），文本型 PDF
- 日期格式 YYYY-MM-DD
- 单卡 622778****7108
- 注意：大文件（>300KB）可能是图片 PDF，无法提取文字

## 数据源
- 文件目录见脚本内默认路径

## 结果: 35/47 个文件 (2022.06~2025.05), 1,826 txns, 100% PASS
